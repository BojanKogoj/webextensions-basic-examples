## Loading

### Firefox

In `about:debugging` click `Load Temporary Add-on`

### Chrome

In `chrome://extensions/` click `Load unpacked extension...`

### Edge

 - In `about:flags` check `Enable extension developer features`
 - Menu > Extensions > Load extension
